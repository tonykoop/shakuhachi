# Photo Shot List — SHK-D4-P1

Twelve canonical shots that feed the README hero, capstone deck, print packet, build-log site, and visual BOM. Shoot in this order so the build-log site reads chronologically.

## Setup

- Camera: phone (iPhone 13 Pro or equivalent) is fine; resolution ≥ 12 MP.
- Lighting: north-facing window during daytime, supplemented by a 5500 K LED soft-box from camera-left at 45°.
- Background: neutral off-white seamless paper for product shots; in-shop context for process shots.
- Color reference: include an X-Rite ColorChecker or grey card in the first frame of each session.
- File naming: `SHK-D4-P1-<step>-<descriptor>.jpg` (e.g. `SHK-D4-P1-04-bore-validation-pine.jpg`).
- Output paths: hero + finished shots → `images/`; process shots → `images/build/`; SVG drawings → `drawings/`.

## Shot list

### 01 — Pre-build (materials)

- **01a.** Three cherry blanks stacked, end-grain visible, with caliper dimensioning the 1×1 face. Filename: `SHK-D4-P1-01-blanks.jpg`. Caption: *"Three quarter-sawn cherry blanks, 1×1×24 in. The deep-bore drilling step is more forgiving on quarter-sawn than flat-sawn stock."*
- **01b.** Tooling group photo: pilot bit + step-up bits + 20 mm reamer + tailstock vise. Filename: `SHK-D4-P1-01-tooling.jpg`. Caption: *"Deep-bore tooling for the 21.5-in cherry bore. Reamer is 24 in OAL."*

### 02 — Fixture build (one-time)

- **02a.** Tailstock vise welded up, side view on workbench with calipers indicating the V-channel internal angle. Filename: `SHK-D4-P1-02-fixture-vise.jpg`. Caption: *"Welded vise: 6 in V-channel, two 3/8-16 side clamps, MT2 shank. Built once; reusable across the family."*
- **02b.** Vise mounted on lathe tailstock with pine scrap clamped, runout dial indicator on body. Filename: `SHK-D4-P1-02-fixture-runout.jpg`. Caption: *"Indicating < 0.005 in runout to spindle axis before any production drilling."*

### 03 — Drilling (process)

- **03a.** Pine scrap mid-bore on the lathe, drill bit advancing from headstock. Filename: `SHK-D4-P1-03-scrap-bore.jpg`. Caption: *"Scrap-stock validation pass — pine, same fixture, same depth as production. Mandatory before each cherry blank."*
- **03b.** Cherry blank mid-step-up bore. Wide shot showing lathe + chip evacuation. Filename: `SHK-D4-P1-03-cherry-bore.jpg`. Caption: *"Stepping up the bore in 4 mm increments. 1/2 → 5/8 → 3/4 → 0.787 in reamer. Pecks ≤ 1.5 in."*
- **03c.** Bore-gauge measurement at the foot end immediately after the reamer pass. Filename: `SHK-D4-P1-03-bore-gauge.jpg`. Caption: *"Post-bore validation: 0.787 in ± 0.005 in at three stations, 120° apart at each station to catch ovalization."*

### 04 — Outside turn (process)

- **04a.** Body mounted between centers (registered off the bore via pin-centers). Filename: `SHK-D4-P1-04-between-centers.jpg`. Caption: *"Outside turn registered off the established bore. Bore axis = lathe rotation axis = acoustic axis."*
- **04b.** Roughing-gouge cut shaping the OD; chips flying. Filename: `SHK-D4-P1-04-roughing.jpg`. Caption: *"Rough to 1.20 in OD, finish to 1.10 in OD with a spindle gouge."*

### 05 — Hole layout and drilling

- **05a.** Paper-wrap template wrapped around the body, taped, with awl punching hole-1 center. Filename: `SHK-D4-P1-05-template.jpg`. Caption: *"Tone-hole template wrapped at 1:1 scale. Punched centers transfer to the body."*
- **05b.** Body in tone-hole jig on drill press, 7 mm brad-point bit at hole 3 station. Filename: `SHK-D4-P1-05-drilling.jpg`. Caption: *"V-cradle jig drills front and back holes from the same axis. All holes start undersize at 7 mm."*

### 06 — Utaguchi cut

- **06a.** Bandsaw rough-cut on the utaguchi notch. Filename: `SHK-D4-P1-06-bandsaw.jpg`. Caption: *"Rough notch — bandsaw, 0.030 in finishing stock left."*
- **06b.** Chisel + 32° gauge paring the inner bevel. Filename: `SHK-D4-P1-06-chisel-gauge.jpg`. Caption: *"32° gauge against the inner bevel. Cut shallow first; deepen during tuning."*

### 07 — Tuning (process)

- **07a.** Body horizontal on bench, tuner reading the fundamental, hands holding the body to blow. Filename: `SHK-D4-P1-07-tuning-fundamental.jpg`. Caption: *"Bisque-style tuning. Fundamental Ro / D4. Pre-tune target ±25 cents; post-tune ±10 cents."*
- **07b.** Diamond round file mid-stroke, opening hole 3. Filename: `SHK-D4-P1-07-filing-hole.jpg`. Caption: *"Hole-tuning workflow: file open in 0.5 mm steps, re-measure, repeat. Never enlarge with the drill."*

### 08 — Finished hero

- **08.** SHK-D4-P1 finished, oblique view at 35°, neutral seamless background, one-light setup. Filename: `SHK-D4-P1-08-hero.jpg`. **This is the README hero image.** Caption: *"SHK-D4-P1 — solid cherry, 21.5 in OAL. Bore drilled with the headstock-driven technique; oil-finished; tuned to D4."*

### 09 — Detail shots

- **09a.** Utaguchi close-up, 4:1 macro, side-light. Filename: `SHK-D4-P1-09-utaguchi.jpg`. Caption: *"Utaguchi: 0.787 in wide, 0.315 in deep, 32° inner bevel."*

- **09c.** Tone-hole cluster, oblique. Filename: `SHK-D4-P1-09-holes.jpg`. Caption: *"Front holes 1–4 (Tsu / Re / Chi / Ri). Each filed to within ±10 cents of target."*
