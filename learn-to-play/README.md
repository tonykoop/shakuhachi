# learn-to-play / Shakuhachi (D)

Starter songbook for the D shakuhachi from the build packet. Generated by the sheet-music skill.

The shakuhachi is an end-blown bamboo flute with a quiet attack, flexible pitch, and expressive meri/kari shading.

Range: **D4 - D6 (two octaves). Five-hole pentatonic core with cross-fingered diatonic notes.**

## Notes on notation
Meri/kari markings are written as comments in the ABC header. Treat grace notes as slow pitch inflections unless a note says otherwise.

## L2 readiness and known gaps
This starter songbook is ready for L2 review as a playable packet draft: ABC melodies, per-tune notes, static fingering summaries, warmups, and a top-level chart are present. Known gaps: generated audio, MusicXML, LilyPond, MIDI, and printable songsheets are skipped until the shared render dependencies are available; fingering charts need builder review against the final hole layout.
